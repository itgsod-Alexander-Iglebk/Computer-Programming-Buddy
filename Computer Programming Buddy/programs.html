PROGRAMMERING SPRÅK

  • HTML/CSS PROGRAMMING
    HTML är ett standard programmering språk som används för att koda hemsidor. HTML kombineras oftast med CSS som är ett verktyg för HTML när det kommer till positionering och design.

    Intresse anmälan: cpbinc@outlook.com

  • JAVASCRIPT PROGRAMMING
    Javascript som är en av världens mest kändaste och använda programmering språk används till att skapa egna program, t.ex. appar eller funktionära system.

    Intresse anmälan: cpbinc@outlook.com

  • RUBY PROGRAMMING
    Ruby är ett programmering språk som är inte lika känd som Javascript men väldigt användbar för nybörjare. Språket är en av de lättaste programmering språken att lära sig för nybörjare.

    Intresse anmälan: cpbinc@outlook.com

  • PYTHON PROGRAMMING
    Python är ett programmering språk som är väldig lik språket Ruby. Detta språk är väldigt lätt lärt för nybörjare.

    Intresse anmälan: cpbinc@outlook.com
